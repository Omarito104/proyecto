Restaurante Especializado En Pescado

El sistema de información está especializado en un restaurante de comidas marinas, como resultado de un estudio de necesidades de atención al cliente en el momento que realiza y se le entrega el pedido de una forma rápida y de buena calidad en la ciudad de bogotá. El Sistema de información ofrece la gestión de ventas de un restaurante llevando un control de los precios de cada producto, la gestión de ventas y facturaciones, consulta del inventario (comida preparada) y la gestión de clientes.

Fundaentos

El propósito de este proyecto es solucionar el problema que presenta el restaurante, creando un sistema de información orientado a la web, que permita gestionar las ventas y tener un registro de clientes. Al implementar este sistema se podrá tener una gestión completa de las ventas, lo cual agilizara todos los procesos en el restaurante.

Objetivo

Desarrollamos un sistema de información, orientado a la web que permita mantener el control de las actividades financieras del restaurante, implementando un sistema que gestione las ventas del pedido solicitado por el cliente, basado en el menú.

Proposito

Este documento tiene el propósito de definir los requerimientos funcionales y los no funcionales del sistema de información orientado a la web que permita mantener el control de las actividades financieras del restaurante, gestionar las ventas y tener un registro de los clientes.


Alcance

El proyecto permite realizar un registro y control de los clientes que ingresan al restaurante, simultáneamente se hará un registro de los artículos vendidos con sus respectivos precios para que el gerente tenga conocimiento de la cantidad de ingresos generados
